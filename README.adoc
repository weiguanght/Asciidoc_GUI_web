[[features]]
== Features

=== Rich Text Editing
* **WYSIWYG Editing**: Intuitive rich text editing experience powered by Tiptap
* **Split View Mode**: Side-by-side source code and preview panels
* **Source Code Mode**: Direct AsciiDoc source editing with syntax highlighting
* **Real-time Preview**: Instant preview of AsciiDoc rendering

=== AsciiDoc Support
* Full AsciiDoc syntax support via Asciidoctor.js
* Headings (H1-H6), paragraphs, and text formatting
* Bold, italic, underline, strikethrough, and highlight
* Ordered and unordered lists
* Blockquotes and admonitions (NOTE, TIP, WARNING, CAUTION, IMPORTANT)
* Code blocks with syntax highlighting (Highlight.js)
* Tables with merge cell support
* Images (drag & drop, paste, URL insertion)
* Links and cross-references
* Mathematical formulas (KaTeX)
* Mermaid diagrams

=== User Interface
* **Dark Mode**: Toggle between light and dark themes
* **Mobile Responsive**: Adaptive layout for mobile devices
* **Collapsible Sidebar**: Document management and settings
* **Floating Toolbar**: Context-aware formatting options
* **Search & Replace**: Find and replace text with regex support

=== File Operations
* Create, open, and save AsciiDoc files
* Export to multiple formats:
** `.adoc` (AsciiDoc source)
** `.html` (Standalone HTML)
** `.zip` (Archive with images)
* Image management with embedded Data URLs

[[getting-started]]
== Getting Started

=== Prerequisites

* Node.js 18+ 
* npm or yarn

=== Installation

[source,bash]
----
# Clone the repository
git clone https://github.com/yourusername/asciidoc-wysiwyg-editor.git

# Navigate to project directory
cd asciidoc-wysiwyg-editor

# Install dependencies
npm install

# Start development server
npm run dev
----

=== Build for Production

[source,bash]
----
# Build the project
npm run build

# Preview production build
npm run preview
----

=== Deploy to GitHub Pages

[source,bash]
----
npm run deploy
----
